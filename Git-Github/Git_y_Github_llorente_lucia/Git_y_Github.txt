<p> A. Git es un software de control de versiones que registra los cambios realizados sobre un archivo/s a lo largo del tiempo. Fue creado por Linus Torvalds en 2005 para ser utilizado en el desarrollo del kernel de Linux. </p>
<p> GitHub actualmente pertenece a Microsoft. Realizo la compra ya que la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios, buscando estar en contacto directo con los desarrolladores, ver sus necesidades y ofrecerles herramientas que faciliten su integración </p>
<p> GitHub utiliza Git para el control de versión, desde la terminal de la computadora Mac, Linux, Microsoft. </p>
